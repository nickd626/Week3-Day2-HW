Syntax for built-in functions
-----------------------------
map(function,iterable)
filter(function,iterable)
zip(*iterables)
reduce(function, iterable)

lambda expresssions
(lambda parameter: expression)
-- Anonymous functions that are only used once.
-- Benefits of lambda expressions are that they save memory.

Recursive Function
-- Needs a Base case & Recursive case
-- The Base case breaks out of the Recursive loop
-- The Recursive case decrements/increments argument

Generator Function
-- the yield keyword denotes a Generator Function
-- Benefit of a Generator is that they save memory.
